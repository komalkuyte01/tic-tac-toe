#  Tic Tac Toe Game

This is a simple and interactive **Tic Tac Toe** game built using **HTML**, **CSS**, and **JavaScript**. Two players take turns marking **X** and **O** on a 3Ã—3 grid. The first player to align three symbols horizontally, vertically, or diagonally wins the game. 

##  Technologies Used
- HTML 
- CSS 
- JavaScript 

##  Features
- 2-player gameplay
- Win and draw detection
- Reset and New Game buttons
- Simple and clean UI

##  How to Run
1. Clone or download this repository.
2. Open the `index.html` file in your browser.
3. Start playing by clicking on the boxes.
